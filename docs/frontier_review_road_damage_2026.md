# Frontier review for Japan7 road-damage detection (2026-07-21)

## Evidence-first conclusion

The current bottleneck is not yet a backbone, neck, head, or loss. The Japan7 train/val split contains visually confirmed near-duplicate road scenes. Until a scene/sequence-grouped split is established, model ranking and high-IoU conclusions are not reliable enough to select a new architecture.

The papers below are therefore retained as conditional candidates, not as reasons to stack modules now.

## Current exclusions

- No full RSDD-style backbone+neck+head stack: it cannot identify which change helps and loses clean pretrained attribution.
- No TOOD transplant: task-aligned assignment is already present.
- No direct GFL/VFL/NWD/uncertainty change: the active loss path and clean-data error profile must be established first.
- No multi-head or PCGrad implementation: the requested gradient evidence does not exist because the formal 100e pair was correctly stopped.
- No more Laplacian/global high-frequency enhancement: prior P3/P4 experiments do not support it, and near-duplicate leakage is the dominant validity issue.
