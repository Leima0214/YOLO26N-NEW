## Interpretation

- D10 is primarily a thin, elongated representation problem: about 90% of D10 boxes have aspect ratio at least 3, while only about 6% are small at image size 640.
- A1/A2 show that changing the box loss does not repair D10. B2 shows that increasing classification pressure does not repair D10 either.
- The next Paper 1 route is still a module paper. It is not a distillation route.

## Tomorrow's Only Authorized Route

1. Implement one P3-only, pretrained-friendly directional residual module: depthwise `3x3`, horizontal strip branch, vertical strip branch, and a gated residual output.
2. Audit construction, identity/near-identity initialization, finite gradients, fused output, and checkpoint transfer before training.
3. Run `1e smoke`, then run the matched `30e signal` only if smoke passes.
4. Promote only at `mAP50-95 >= 0.323`; treat `0.319-0.322` as neutral and reject below `0.319`.
5. Only after a valid directional signal, implement a shallow P2-to-P3 detail injection that preserves the original P3/P4/P5 Detect heads. Do not add an old CPUBone P2Lite head or restart EMA.
6. A final three-module model is allowed only after the directional and shallow-detail modules have evidence. The third module should be an adaptive fusion gate within that shallow-detail path, not a recycled negative module.
